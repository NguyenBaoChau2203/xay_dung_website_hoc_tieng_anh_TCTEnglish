# XÂY DỰNG WEBSITE HỖ TRỢ HỌC TỪ VỰNG TIẾNG ANH CHO CÔNG TY TCT

## 📌 Giới thiệu
Dự án được xây dựng nhằm hỗ trợ nhân viên công ty TCT học và ôn tập từ vựng tiếng Anh theo các chủ đề công việc, giúp nâng cao khả năng sử dụng tiếng Anh trong môi trường làm việc.

Đây là **đồ án kết thúc môn Lập trình Web**, được thực hiện bởi nhóm sinh viên ngành Công nghệ Thông tin.

---

## 🛠 Công nghệ sử dụng
- **Framework:** ASP.NET MVC (.NET 10)
- **Ngôn ngữ:** C#
- **Cơ sở dữ liệu:** Microsoft SQL Server
- **IDE:** Visual Studio
- **Quản lý mã nguồn:** Git & GitHub

---

## 📂 Chức năng chính
- Quản lý người dùng (Đăng ký / Đăng nhập)
- Học từ vựng theo chủ đề
- Luyện tập và kiểm tra từ vựng
- Theo dõi tiến độ học tập
- Quản trị nội dung từ vựng (Admin)

## 🚀 Hướng dẫn chạy dự án
1. Clone repository:
git clone https://github.com/NguyenChau/xay_dung_website_hoc_tu_vung_tieng_anh_TCT.git

2. Mở project bằng **Visual Studio**

3. Cấu hình chuỗi kết nối SQL Server trong:
appsettings.json

4. Chạy project:
F5
